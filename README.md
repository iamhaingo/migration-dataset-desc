# migration-dataset-desc
A repo for description of migration dataset used for our project

- Displacement data, estimates of internally displaced persons due to climate-related events or conflict: https://www.internal-displacement.org/database/

- Landsat 9 Imagery, 30-meter resolution satellite imagery to capture environmental feature: https://developers.google.com/earth-engine/datasets/catalog/landsat-9

- NASA Black Marble (Nighttime Lights): Data from the VIIRS Day/Night Band (VNP46A2) is used to track nighttime radiance at a 500-meter resolution: https://developers.google.com/earth-engine/datasets/catalog/NASA_VIIRS_002_VNP46A2

- Meta data : full description of data from Meta can be found in the directory ==meta-data-description==
https://ai.meta.com/ai-for-good/datasets/?filter=mobility
